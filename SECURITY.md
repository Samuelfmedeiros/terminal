# 🔒 Segurança — Terminal (Samuel Medeiros)

## Reportando Vulnerabilidades

Se você descobrir uma vulnerabilidade de segurança neste repositório, por favor reporte através de:

- **Email:** samuelandrademedeiros@gmail.com
- **GitHub Security Advisory:** https://github.com/Samuelfmedeiros/terminal/security/advisories

Tentaremos responder em até 48 horas.

## Práticas de Segurança

- ✅ **Gitleaks** — varredura de secrets ativa via CI
- ✅ **Dependabot** — atualizações automáticas de dependências
- ✅ **Push Protection** — GitHub bloqueia push de tokens/credenciais
- ❌ Nenhum arquivo `.env`, `audio_*`, ou credencial trackeada

## Política de Divulgação

Por favor, NÃO abra issues públicas para reportar vulnerabilidades de segurança. Use os canais privados acima.
